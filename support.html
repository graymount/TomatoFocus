❓ 常见问题

如何设置自定义时间？

1. 在主界面点击"预设"按钮

2. 选择"专注预设"或"休息预设"中的选项

3. 或者使用快捷按钮（+5分钟、+10分钟、-5分钟）调整时间

4. 也可以点击时钟图标选择不同模式

如何切换主题？

1. 在主界面右上角点击"主题"按钮

2. 每次点击都会切换到下一个主题

3. 包含森林、海洋、咖啡厅、极简、Lofi、科技六种主题

4. 主题会影响界面颜色和背景音效

背景音效如何使用？

1. 在界面底部找到音效控制面板

2. 点击音效名称选择不同的背景声音

3. 使用播放/暂停按钮控制音效

4. 拖动音量滑块调整音量大小

5. 包含森林、雨声、咖啡厅等多种环境音效

专注模式有什么用？

1. 点击右上角的"专注"按钮进入沉浸模式

2. 在此模式下界面更简洁，减少干扰

3. 只显示计时器和基本控制按钮

4. 适合需要高度专注的工作时段

统计数据在哪里查看？

1. 点击底部标签栏的"统计"选项

2. 查看今日、本周、本月的专注时间

3. 查看专注天数和平均专注时长

4. 查看不同模式（专注、短休息、长休息）的使用情况
